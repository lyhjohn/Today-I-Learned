# [MySQL] 윈도우 명령프롬프트로 mysql 접속하기

### 윈도우 명령프롬프트로 mysql 명령어를 이용해 db에 접속하려면 환경변수 설정이 되어야 한다.

### 그동안 귀찮아서 MySQL 커맨더로 접속했는데, 자꾸만 신경이 쓰여서 이참에 환경변수 값을 설정하기로 했다.
